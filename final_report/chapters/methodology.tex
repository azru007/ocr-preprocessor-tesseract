\chapter{Methodology}
\section{DBNet model inference}
The preprocessing pipeline begins with native inference using the DBNet model. The ONNX Runtime executes the quantized model on the mobile CPU, producing a high-resolution probability map representing the exact localized text regions.

\section{Geometric unwarping and stabilization}
Detected contours are processed through the Vatti clipping algorithm to define stable bounding polygons. A perspective transform matrix is then calculated to rectify skewed text lines, providing a normalized input for the OCR engines.

\section{Integral image accelerated binarization}
To handle complex lighting gradients, we apply Sauvola adaptive thresholding. This process is optimized using Integral Images (Summed Area Tables), allowing for constant-time local mean and variance calculations.

\section{JNI and Android system integration}
The core logic is integrated via a zero-copy JNI layer. By using locking the Android Bitmap memory directly in native C++, the engine modifies the image data in-place, eliminating the overhead of buffer copies.
